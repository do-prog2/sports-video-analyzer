# Sports Video Analyzer

## Project Overview

The Sports Video Analyzer is a program designed to analyze sports videos. It provides various features including automated highlights extraction, performance metrics, and video playback analysis to enhance the understanding of sports events.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/do-prog2/sports-video-analyzer.git
   cd sports-video-analyzer
   ```

2. **Install dependencies**:
   Ensure you have Python 3.x installed, then use pip to install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the program**:
   Use the following command to start analyzing your sports videos:
   ```bash
   python main.py --video <path_to_video>
   ```

4. **Configuration**:
   Modify the `config.json` file to adjust settings and parameters as needed for your analysis.

5. **Contributions**:
   Contributions are welcome! Please create a pull request for any modifications or enhancements.
